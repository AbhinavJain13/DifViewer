DifViewer
=========

Renders .dif files (MB Specification). Also exports to wavefront .obj.

Rendering
---------

Run as `DifViewer <file>` to show the renderer

Platforms Supported
-------------------

Mac OS 10.7+

Exporting
---------

To export, run as `DifViewer -o <obj file> <dif file>` to convert <dif file> into <obj file>
